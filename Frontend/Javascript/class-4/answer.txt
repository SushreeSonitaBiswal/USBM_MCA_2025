1. What does Math.floor() do in JavaScript?
 In javascript Math.floor() is a function that returns the largest integer less 
 than or equal to the given number.

 Example:
 Math.floor(5.7) //5
  Math.floor(3.2)//3
   Math.floor(-2.5)//-3
